Auto B.O.M. Filler - Privacy Policy

Last updated: March 6, 2026

What data we collect

Auto B.O.M. Filler collects no personal data. The extension only processes:

  • Product information (name, price, weight) from web pages you explicitly right-click on
  • Your Google Sheets spreadsheet ID and sheet name (stored locally in Chrome sync storage)

How data is used

Product data extracted from web pages is sent only to your own Google Sheets spreadsheet via the Google Sheets API. No data is sent to any third-party server.

Google API usage

The extension uses Google Identity API to authenticate with your Google account and Google Sheets API to write data to your spreadsheets. This requires the spreadsheets OAuth scope. Your Google auth token is managed by Chrome and never stored or transmitted by the extension.

Data storage

Project settings (spreadsheet ID, sheet name) are stored in Chrome's sync storage. No data is stored on external servers.

Third parties

No data is shared with or sold to third parties. Ever.
